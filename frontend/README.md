## 🛠️ How to Run
For the integrated production environment:
1.  Go to the `backend` folder.
2.  Run `npm run dev`.

For frontend development only:
1.  Run `npm run dev` in this folder (`frontend`).
2.  Access at [http://localhost:5173](http://localhost:5173).
